# 酒馆素材库

酒馆素材库 是一个面向 **SillyTavern（ST）相关素材** 的本地 Windows 桌面管理器。

它用于整理、浏览和批量管理角色卡、世界书、预设、剧情推进预设、数据库表格模板、辅助脚本、正则、美化与元数据覆盖等资产。

## 当前版本

- 平台：Windows
- 界面：PySide6
- 版本：`v1.0.0`
- 作者：**超级花火大王**

## 功能

- 导入文件 / 文件夹
- 按类型浏览资产
- 搜索资产
- 查看资产详情
- 预览角色卡 PNG
- 手动改类型 / 待复核处理
- 批量勾选、批量删除、批量导出
- 重命名资产

## 当前支持的分类

1. 角色卡
2. 世界书
3. 预设
4. 剧情推进预设
5. 数据库表格模板
6. 辅助脚本
7. 正则
8. 美化
9. 元数据覆盖
10. 待复核

## 使用方式

### 运行源码

```bash
python gui_main.py
```

或：

```bash
python main.py gui
```

### 使用打包产物

- 目录版：`dist/STRepertory.exe`
- 安装版：`build/installer/output/STRepertory-setup.exe`

## 基本操作

- 点击资产行：查看详情和预览
- 点击“批量”列：加入或取消批量操作集合
- 支持全选 / 清空勾选
- 批量操作包括删除与导出
- 右侧操作区支持重命名、改类型和复核
